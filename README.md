# Gifty (Flutter)

Premium mobile-first social gifting app scaffold built with Flutter + Riverpod + Material 3.

## Highlights
- Dark premium design system with purple/pink gradients
- Feature-first architecture (auth, calendar, wishlist, qr, groups, notifications, profile)
- Localization-ready foundation (en, sk, cs, de, pl, hu)
- Mock-first data layer with clean extension points for Supabase, realtime and push notifications
- iOS/Android deployment-ready project layout
